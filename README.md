# 🚀 JobPilot AI

A smart web application to track job applications, analyze progress, and improve job search efficiency.

Built using Python and Streamlit.

---

## 🌟 Features

• Secure user login and registration  
• Track job applications  
• Dashboard with application statistics  
• AI job description match analyzer  
• Job search integration  
• Interview reminders  
• Application analytics charts  
• CSV export of job data  

---

## 🛠 Tech Stack

Python  
Streamlit  
SQLite  
Pandas  
Scikit-Learn  
Matplotlib  

---

## 📊 Dashboard

The dashboard provides a quick overview of:

- Total applications
- Interview calls
- Job offers
- Rejections

---

## 🤖 AI Job Match

Uses TF-IDF vectorization and cosine similarity to compare:

```
User skills
VS
Job description
```

and generates a **match score**.

---

## 🔍 Job Search

Fetches job listings from public APIs and allows users to explore opportunities directly within the platform.

Install dependencies:

```

```

Run the app:

```
streamlit run app.py
```
